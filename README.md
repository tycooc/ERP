# ERP

ERP系统是企业资源计划(Enterprise Resource Planning )的简称，是指建立在信息技术基础上，集信息技术与先进管理思想于一身，以系统化的管理思想，为企业员工及决策层提供决策手段的管理平台。

# 技术框架

* JS框架：Jquery 1.8.0
* UI框架: EasyUI 1.3.5
* 桌面框架: HoorayOS
* 核心框架：Spring 3.0.5
* 视图框架：Struts 2.2.3
* 持久层框架：Hibernate 3.0
* 日志管理：Log4j 1.2.16
* 项目管理框架: Maven3.2.3

# 开发环境

* IDE: IntelliJ IDEA2016
* DB: Mysql5.1
* JDK: JDK1.8
* WEB: Tomcat7.0
* Maven: Maven3.2.3

# 运行环境

* WEB服务器：Tomcat7.0
* 数据库服务器：Mysql5.1
* JAVA平台: JRE1.8
* 操作系统：Windows7
